# Jatin's Motion Design Portfolio

A modern, interactive portfolio website showcasing video editing and motion design work. Built with React, Framer Motion, and Tailwind CSS.

##  Features

- **Hero Section**: Eye-catching introduction with smooth animations
- **Portfolio Gallery**: Organized project showcase
- **Motion Graphics**: Local video playback with autoplay and loop
- **About Section**: Personal branding with interactive effects
- **Responsive Design**: Mobile-first, fully responsive layout
